## API CONTRACTS

### Create Project Admin
```
POST /api/auth/masteradmin/admin-users/create-project-admin/

Request:
{
  "project_id": 1,
  "admin_type": "client",
  "username": "client_admin",
  "company_name": "ABC Corp",
  "registered_address": "123 Main St, City, State - 12345"
}

Response (201 Created):
{
  "username": "client_admin",
  "password": "aB3$xY9@mN2pQ5!z",
  "admin_type": "client",
  "company_name": "ABC Corp",
  "registered_address": "123 Main St, City, State - 12345"
}

Errors:
400 - Validation error (username exists, missing fields, invalid project)
404 - Project not found or access denied
```

### Get Project Admins
```
GET /api/auth/masteradmin/projects/{project_id}/admins/

Response (200 OK):
{
  "client": {
    "id": 1,
    "username": "client_admin",
    "admin_type": "client",
    "company_name": "ABC Corp",
    "registered_address": "123 Main St",
    "project": 1,
    "project_name": "Solar Project",
    "is_active": true,
    "created_at": "2025-02-06T10:00:00Z"
  },
  "epc": null,
  "contractors": []
}

Errors:
400 - User not associated with tenant
404 - Project not found
```

6. **One-time Password Display** - Password returned only once in API response

## VALIDATION RULES

### Username
- Required
- No spaces allowed
- Must be unique across all users
- Alphanumeric + underscore only

### Company Name
- Required
- Max 255 characters

### Registered Address
- Required
- No max length (TextField)

### Project ID
- Required
- Must exist in database
- Must belong to master admin's tenant

### Admin Type
- Required
- Must be one of: client, epc, contractor

---

## PASSWORD POLICY

### Generation
- Length: 16 characters
- Character set: `string.ascii_letters + string.digits + '!@#$%^&*'`
- Method: `random.choices()` for cryptographic randomness
- Example: `aB3$xY9@mN2pQ5!z`

### Flags
- `is_autogenerated_password = True`
- `is_password_reset_required = True`

### First Login
User must reset password on first login (enforced by backend).
