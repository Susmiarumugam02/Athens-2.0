## 🔌 API DOCUMENTATION

### Create Project Admin

**Endpoint:** `POST /api/auth/masteradmin/admin-users/create-project-admin/`

**Request:**
```json
{
  "project_id": 1,
  "admin_type": "client",
  "username": "client_admin",
  "company_name": "ABC Corporation",
  "registered_address": "123 Main Street, City, State - 12345"
}
```

**Response (201 Created):**
```json
{
  "username": "client_admin",
  "password": "aB3$xY9@mN2pQ5!z",
  "admin_type": "client",
  "company_name": "ABC Corporation",
  "registered_address": "123 Main Street, City, State - 12345"
}
```

**Errors:**
- `400` - Validation error (username exists, missing fields, invalid project)
- `404` - Project not found or access denied

### Get Project Admins

**Endpoint:** `GET /api/auth/masteradmin/projects/{project_id}/admins/`

**Response (200 OK):**
```json
{
  "client": {
    "id": 1,
    "username": "client_admin",
    "admin_type": "client",
    "company_name": "ABC Corporation",
    "registered_address": "123 Main Street",
    "project": 1,
    "project_name": "Solar Power Plant",
    "is_active": true,
    "created_at": "2025-02-06T10:00:00Z"
  },
  "epc": null,
  "contractors": []
}
```
